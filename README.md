# Splunk SIEM Log Analysis Project

**This repository features a set of hands on projects focused on analyzing different types of logs using _Splunk SIEM_. Each project walks through the complete workflow from ingesting sample log data to extracting insights and identifying potential security issues.**

## Included Projects

# 1. [DNS Log Analysis](./DNS%20Log%20Analysis/DNS-Log-Analysis-using-SIEM.md)

**A step-by-step guide to analyzing Domain Name System (DNS) logs in Splunk.**

*What it covers :*

    Upload and parse DNS log data
    Extract key fields
    Identify query patterns
    Detect anomalies and suspicious activity
    Monitor DNS Traffic effectively
